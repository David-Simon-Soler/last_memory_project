Last Memory (Story) - Plataforma de Legado Emocional

Este repositorio contiene la documentación estratégica y financiera del proyecto "Last Memory", galardonado con el Premio Innovación Social (1.000€) en el IX Triatlón de Emprendimiento Social de la Universidad de Alicante (Mayo 2026).

1. Resumen del Proyecto

Last Memory es una solución basada en Inteligencia Artificial diseñada para preservar el legado emocional y la memoria histórica personal de las personas. A través de un chatbot interactivo y almacenamiento seguro, permitimos que las familias mantengan viva la conexión emocional incluso después de una pérdida.


2. Análisis de Negocio y Métricas
Como parte fundamental del proyecto, se desarrolló un plan de viabilidad con enfoque en análisis de datos:

* Punto de Equilibrio (Break-even): Estimado entre el mes 16 y 18.

* Proyecciones Financieras: Previsión de 500.000€ en ingresos recurrentes (MRR) para el año 4.

* Eficiencia de Marketing: Objetivo de CPA (Coste de Adquisición) inferior a 25€.

* Retención de Clientes: Meta superior al 75%.


3. Stack Tecnológico Propuesto

El despliegue del proyecto contempla:
  
* IA/Chatbots: Modelos de lenguaje personalizados capaces de interactuar basados en archivos de audio, fotos y cartas.
  
* Infraestructura Cloud: Almacenamiento escalable.
